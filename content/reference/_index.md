**Reference is information.** It is the technical description of the machinery of
Epimethean Challenge — order formats, unit and ship statistics, technology trees,
turn-processing rules, and the game's data structures.

Reference material is meant to be *consulted*, not read cover to cover. It is
accurate, complete, and neutral: it describes how the game works without teaching
or persuading. When you need to know exactly what a command does or what a value
is, look here.
